# Feature 001. Публичная главная и навигация

## Назначение
Реализовать публичную витрину сайта с глобальной шапкой, меню, футером, быстрыми переходами по основным разделам, промо-блоками, CTA и entry points в каталог, новости, комьюнити и партнерские сценарии.

## Покрываемые маршруты
- `/`
- `/home`
- `/community`

## Основной функционал
- Отрисовка hero и промо-блоков на базе CMS-конфигурации.
- Глобальная навигация по контентным, каталоговым и партнерским разделам.
- Поиск входов в каталог, benefits, регистрацию и личные кабинеты.
- Персонализация блоков для гостя и авторизованного пользователя.
- Адаптивная верстка для desktop/tablet/mobile.

## Роли
- Гость
- Авторизованный пользователь
- Контент-менеджер как поставщик контента
