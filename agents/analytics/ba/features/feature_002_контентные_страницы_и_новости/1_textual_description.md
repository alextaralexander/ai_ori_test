# Feature 002. Контентные страницы и новости

## Назначение
Воспроизвести контентный контур сайта для публикации новостей, промо-материалов и отдельных лендинговых страниц с динамическим идентификатором контента.

## Покрываемые маршруты
- `/news`
- `/content/:contentId`
- `/offer/:offerId`

## Основной функционал
- Лента новостей с карточками, баннерами и переходом в детали.
- Динамические контентные страницы по `contentId` и промо-предложения по `offerId`.
- Поддержка изображений, rich text, CTA, встроенных PDF и ссылок на продукты.
- Публикация запланированного контента и снятие с публикации.
- SEO-поля, breadcrumbs и переиспользуемые шаблоны контента.

## Роли
- Гость
- Авторизованный пользователь
- Контент-менеджер
