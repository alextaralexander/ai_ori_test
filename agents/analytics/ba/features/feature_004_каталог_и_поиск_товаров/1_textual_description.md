# Feature 004. Каталог и поиск товаров

## Назначение
Реализовать поиск и каталог товаров с фильтрами, сортировками, тегами, выдачей по запросу и связкой с корзиной.

## Покрываемые маршруты
- `/search`

## Основной функционал
- Поиск по названию, коду, категории и ключевым словам.
- Фильтрация по атрибутам, цене, доступности, тегам и промо-меткам.
- Сортировка, пагинация и lazy loading карточек.
- Добавление товара в корзину прямо из списка.
- Поддержка пустой выдачи, рекомендаций и повторного поиска.

## Роли
- Гость с ограниченной видимостью
- Авторизованный покупатель/партнер
- Категорийный менеджер как владелец каталога
