# Feature 007. Бьюти- и бизнес-бенефиты

## Назначение
Воссоздать лендинги бенефитов для разных типов привлечения с поддержкой партнерского/referral-кода в URL.

## Покрываемые маршруты
- `/beauty-benefits`
- `/beauty-benefits/:code`
- `/business-benefits`
- `/business-benefits/:code`

## Основной функционал
- Показ ценностного предложения для beauty и business сценариев.
- Поддержка персонализированного инвайт-кода в URL.
- CTA в регистрацию, активацию и коммуникацию со спонсором.
- A/B-готовность для разных наборов блоков и офферов.
- Трекинг конверсии по типу лендинга и коду.

## Роли
- Гость
- Потенциальный партнер
- Маркетинг-менеджер
