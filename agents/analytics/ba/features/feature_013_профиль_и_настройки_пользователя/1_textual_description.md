# Feature 013. Профиль и настройки пользователя

## Назначение
Реализовать личные настройки пользователя по основным, контактным, адресным, документным и security-данным.

## Покрываемые маршруты
- `/profile-settings`
- `/profile-settings/general`
- `/profile-settings/contacts`
- `/profile-settings/addresses`
- `/profile-settings/documents`
- `/profile-settings/security`

## Основной функционал
- Просмотр и редактирование личных данных.
- Управление телефонами, email, адресами и документами.
- Смена пароля и security-настроек.
- Валидация обязательных данных для order/claim flows.
- История изменений критичных полей.

## Роли
- Авторизованный пользователь
- Оператор поддержки
