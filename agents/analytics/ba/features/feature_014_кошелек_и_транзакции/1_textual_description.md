# Feature 014. Кошелек и транзакции

## Назначение
Поднять кошелек пользователя с историей транзакций по типам, остатками и движениями средств.

## Покрываемые маршруты
- `/profile/transactions/:type`

## Основной функционал
- Просмотр баланса и транзакций выбранного типа.
- Фильтрация по периоду, статусу и источнику начисления/списания.
- Связка транзакции с заказом, бонусом или корректировкой.
- Экспорт истории и детализация движения.
- Показ pending/posted/reversed операций.

## Роли
- Авторизованный пользователь
- Финансовый оператор
