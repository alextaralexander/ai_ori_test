# Feature 015. Отчеты и квитанции партнера

## Назначение
Повторить отчетный раздел партнера с квитанциями, историей заказов и выгрузкой отчетных документов.

## Покрываемые маршруты
- `/report/info-reciept`
- `/report/order-history`

## Основной функционал
- Просмотр информационных квитанций и order reports.
- Фильтрация по периоду и кампании.
- Скачивание печатных форм и PDF.
- Показ агрегированных итогов по заказам.
- Доступ к отчетам только в разрешенных ролях и периодах.

## Роли
- Авторизованный партнер
- Финансовый/операционный менеджер
