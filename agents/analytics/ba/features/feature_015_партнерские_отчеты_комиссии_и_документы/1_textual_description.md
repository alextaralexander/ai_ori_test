# Feature 015. Партнерские отчеты, комиссии и документы

## Назначение
Реализовать отчетный раздел партнера с комиссионными начислениями, актами, квитанциями, сверкой заказов и документами по выплатам.

## Покрываемые маршруты
- `/report/info-reciept`
- `/report/order-history`

## Основной функционал
- Просмотр отчетов по заказам, продажам структуры и бонусным начислениям.
- Формирование квитанций, актов, справок и сопроводительных документов по выплатам.
- История комиссий, удержаний, выплат и финансовых сверок.
- Фильтрация по периоду, каталогу, заказу, типу документа и бонусной программе.
- Экспорт в PDF/Excel и печатные формы.

## Роли
- Партнер
- Бухгалтер
- Финансовый контролер
