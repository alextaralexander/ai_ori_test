# Feature 016. MLM-структура и рост партнера

## Назначение
Реализовать бизнес-раздел партнера с обзором MLM-структуры, активности команды, квалификаций, карьерного роста и показателей сети.

## Покрываемые маршруты
- `/business`
- `/business/beauty-community`
- `/business/conversion`
- `/business/team-activity`
- `/business/upgrade`
- `/business/partner-card/:personNumber`

## Основной функционал
- Dashboard по ключевым MLM-метрикам, состоянию структуры и целям периода.
- Отчеты по конверсии, активности команды, квалификациям и условиям апгрейда.
- Просмотр карточки партнера по `personNumber` с ролями, статусами и прогрессом.
- Сегментация downline и drill-down в показатели по веткам, уровням и кампаниям каталога.
- Переходы в партнерские заказы, бонусные сценарии и supply-операции.

## Роли
- Партнер-лидер
- Бизнес-менеджер
- MLM-аналитик
