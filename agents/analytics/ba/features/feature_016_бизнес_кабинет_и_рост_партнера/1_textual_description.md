# Feature 016. Бизнес-кабинет и рост партнера

## Назначение
Реализовать бизнес-раздел партнера с обзором развития, конверсии, активности команды, апгрейда и карточек партнеров.

## Покрываемые маршруты
- `/business`
- `/business/beauty-community`
- `/business/conversion`
- `/business/team-activity`
- `/business/upgrade`
- `/business/partner-card/:personNumber`

## Основной функционал
- Dashboard по growth-метрикам и состоянию структуры.
- Отчеты по конверсии, активности команды и условиям апгрейда.
- Просмотр карточки партнера по `personNumber`.
- Сегментация downline и drill-down в показатели.
- Переходы в VIP, supply и order-management сценарии.

## Роли
- Партнер-лидер
- Бизнес-менеджер
