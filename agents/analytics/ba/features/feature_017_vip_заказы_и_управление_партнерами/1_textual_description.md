# Feature 017. VIP-заказы и управление партнерами

## Назначение
Поднять контур VIP order management, включая список партнерских заказов и детальную работу с заказом конкретного партнера.

## Покрываемые маршруты
- `/vip-orders`
- `/business/tools/order-management/vip-orders/partner-orders`
- `/business/tools/order-management/vip-orders/partner-orders/:orderId`

## Основной функционал
- Список VIP/partner orders с фильтрами и поиском.
- Открытие детальной карточки заказа партнера.
- Просмотр оплаты, состава, статусов и связанных событий.
- Поддержка сервисных действий в разрешенных ролях.
- Связка с карточкой партнера и order history.

## Роли
- Партнер-лидер
- Бизнес-оператор
- Сотрудник поддержки VIP-заказов
