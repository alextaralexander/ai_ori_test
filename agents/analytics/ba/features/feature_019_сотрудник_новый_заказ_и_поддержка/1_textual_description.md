# Feature 019. Сотрудник: новый заказ и поддержка

## Назначение
Реализовать employee-контур для создания нового заказа от имени клиента/партнера и для сценариев order support.

## Покрываемые маршруты
- `/employee`
- `/employee/new-order`
- `/employee/order-support`

## Основной функционал
- Рабочее место сотрудника с поиском партнера/клиента.
- Создание нового заказа в операторском режиме.
- Поддержка проблемного заказа, корректировок и консультаций.
- Контроль разрешений и audit trail действий сотрудника.
- Переходы в историю заказов, претензии и карточку партнера.

## Роли
- Сотрудник call-center/backoffice
- Супервизор
