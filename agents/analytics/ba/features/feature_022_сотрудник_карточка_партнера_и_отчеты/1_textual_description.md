# Feature 022. Сотрудник: карточка партнера и отчеты

## Назначение
Поднять employee-функции просмотра карточки партнера и отчетности по его заказам/активности.

## Покрываемые маршруты
- `/employee/partner-card`
- `/employee/report/order-history`

## Основной функционал
- Поиск и открытие карточки партнера.
- Просмотр основных данных, статуса и ключевых KPI партнера.
- Доступ к отчету по истории заказов партнера.
- Переходы в связанные order, claim и support flows.
- Ограничение доступа по роли и зоне ответственности сотрудника.

## Роли
- Сотрудник backoffice
- Региональный менеджер
