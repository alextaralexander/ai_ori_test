# Feature 025. Уведомления, офлайн, i18n и аналитика

## Назначение
Выделить сквозную платформенную фичу для пользовательских уведомлений, offline handling, локализации и подключенных аналитических/маркетинговых пикселей.

## Обнаруженные признаки
- В root-бандле присутствуют offline popup, notification root, i18n namespaces, Yandex Metrika, Mindbox, Hybrid pixel.

## Основной функционал
- Глобальные toast/modal notifications.
- Offline popup и восстановление после reconnect.
- Централизованный i18n-слой и словари переводов.
- Клиентская аналитика и маркетинговые пиксели с событиями pageview/conversion.
- Consent-ready архитектура для трекинга и cookie preferences.

## Роли
- Все роли платформы
- Маркетинг-аналитик
- Администратор трекинга
