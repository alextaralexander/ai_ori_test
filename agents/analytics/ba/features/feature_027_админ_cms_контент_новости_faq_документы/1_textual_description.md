# Feature 027. Админ: CMS, контент, новости, FAQ, документы

## Назначение
Поднять админку для управления всеми контентными сущностями публичного сайта: страницы, новости, FAQ, документы, offer pages.

## Поддерживаемые пользовательские фичи
- `002`, `003`, частично `001`

## Основной функционал
- CRUD для news/articles/content pages/documents.
- WYSIWYG или block-based editor для rich content.
- Планирование публикации, архивирование и versioning.
- Управление SEO, метаданными, вложениями и связями с продуктами.
- Превью перед публикацией и moderation workflow.

## Роли
- Контент-администратор
- Редактор
- Юридический reviewer
