# Feature 029. Админ: PIM, каталог, товары, категории, медиа

## Назначение
Поднять полноценную PIM-админку для управления продуктовым каталогом и всеми данными карточек товара.

## Поддерживаемые пользовательские фичи
- `004`, `005`, частично `001`, `002`

## Основной функционал
- CRUD для товаров, категорий, брендов, атрибутов и тегов.
- Управление кодами товаров, составом, описаниями, ограничениями и статусами.
- Загрузка и версия медиа, PDF, вложений и инструкций.
- Настройка рекомендаций, cross-sell и merchandising blocks.
- Массовый импорт/экспорт каталога.

## Роли
- Категорийный администратор
- Контент/PIM-менеджер
