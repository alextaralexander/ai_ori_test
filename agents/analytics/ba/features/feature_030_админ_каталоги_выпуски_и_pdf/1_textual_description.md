# Feature 030. Админ: каталоги, выпуски и PDF

## Назначение
Создать административный модуль для ведения цифровых каталогов по периодам, выпусков и связанных PDF-материалов.

## Поддерживаемые пользовательские фичи
- `006`, частично `003`

## Основной функционал
- Создание каталога кампании и его статуса публикации.
- Привязка PDF-файлов, изображений страниц и product hotspots.
- Управление текущим и следующим выпуском.
- Архив старых каталогов и schedule публикации.
- Контроль корректности ссылок из каталога в продуктовые страницы.

## Роли
- Каталожный менеджер
- Контент-администратор
