# Feature 032. Админ: WMS/1C, остатки, склады и поставки

## Назначение
Создать административный контур для управления остатками товаров, складами, поставками и синхронизацией запасов из WMS и 1С.

## Поддерживаемые пользовательские фичи
- `005`, `010`, `018`, частично `017`

## Основной функционал
- Учет остатков по складам, каналам продаж и точкам выдачи.
- Управление поставками, supply orders, inbound и outbound перемещениями.
- Резервирование товара под заказ, release reservation и контроль аллокации.
- Правила недоступности, backorder, частичной отгрузки и обработки расхождений.
- Интеграция со складской WMS-системой и 1С как источником остатков и движения товаров.

## Роли
- Логистический администратор
- Складской оператор
- Оператор интеграции складов
