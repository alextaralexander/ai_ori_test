# Feature 032. Админ: остатки, склады, поставки и логистика

## Назначение
Создать административный контур для управления остатками товаров, складами, поставками, supply-цепочками и логистическими статусами.

## Поддерживаемые пользовательские фичи
- `005`, `010`, `018`, частично `017`

## Основной функционал
- Учет остатков по складам и каналам.
- Управление поставками, supply orders и статусов логистики.
- Резервирование товара под заказ и release reservation.
- Правила недоступности, backorder и частичной отгрузки.
- Интеграция со складской/WMS системой.

## Роли
- Логистический администратор
- Складской оператор
