# Feature 034. Админ: претензии, возвраты и сервис

## Назначение
Создать service console для обработки претензий, возвратов, замен, компенсаций и SLA сервиса.

## Поддерживаемые пользовательские фичи
- `012`, `021`

## Основной функционал
- Очередь претензий и операторский workflow.
- Маршрутизация кейса по типу проблемы и решению.
- Управление компенсациями, заменами и возвратами.
- Ведение переписки, вложений и причин отказа/одобрения.
- SLA, приоритеты, escalations и операционная аналитика.

## Роли
- Service admin
- Оператор претензий
- Супервизор сервиса
