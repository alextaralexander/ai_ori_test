# Feature 035. Админ: пользователи, партнеры, сотрудники и имперсонация

## Назначение
Поднять административный модуль управления master-данными людей и организаций, включая партнеров, сотрудников, sponsor links и impersonation controls.

## Поддерживаемые пользовательские фичи
- `008`, `016`, `017`, `019`, `022`, `023`, `024`

## Основной функционал
- Поиск и карточки пользователей, партнеров, сотрудников и sponsor relationships.
- Управление статусом аккаунта, сегментами и eligibility rules.
- Настройка impersonation policy и контроль super-user доступа.
- Изменение данных партнера, связей с офисом и команды.
- Полный audit trail всех операторских действий с персональными данными.

## Роли
- Суперадмин
- Customer master-data admin
- Security admin
