# Feature 036. Админ: KPI, аудит и интеграции платформы

## Назначение
Создать административный аналитический слой с KPI-отчетами, аудитом действий и конфигурацией внешних интеграций платформы Best Ori Gin.

## Поддерживаемые пользовательские фичи
- `015`, `016`, `018`, `025`, частично все transactional features

## Основной функционал
- KPI dashboards по продажам, конверсии, заказам, поставкам, доставке, бонусам и claims.
- Аудит пользовательских, партнерских и операторских действий.
- Конфигурация и мониторинг интеграций с WMS/1С, конвейером сборки, delivery, payment, bonus и analytics системами.
- Экспорт отчетов, плановые выгрузки и SLA-отчеты по интеграциям.
- Alerting по сбоям интеграций и аномалиям ключевых метрик.

## Роли
- Бизнес-администратор
- Data/BI analyst
- Integration admin
