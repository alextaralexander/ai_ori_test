# Feature 036. Админ: отчеты, KPI, аудит и интеграции

## Назначение
Создать административный аналитический слой с KPI-отчетами, аудитом действий и конфигурацией внешних интеграций платформы.

## Поддерживаемые пользовательские фичи
- `015`, `016`, `018`, `025`, частично все transactional features

## Основной функционал
- KPI dashboards по продажам, конверсии, заказам, supply и claims.
- Аудит пользовательских и операторских действий.
- Конфигурация и мониторинг интеграций CRM, ERP, WMS, payment, analytics.
- Экспорт отчетов и плановые выгрузки.
- Alerting по сбоям интеграций и аномалиям ключевых метрик.

## Роли
- Бизнес-администратор
- Data/BI analyst
- Integration admin
