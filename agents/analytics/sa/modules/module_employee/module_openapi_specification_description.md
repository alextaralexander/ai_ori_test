# Module OpenAPI specification description. Employee

Модуль employee публикует Swagger group `employee`, OpenAPI JSON `/v3/api-docs/employee` и Swagger UI `/swagger-ui/employee`. Endpoint-ы находятся под `/api/employee` и внутри package prefix `com.bestorigin.monolith.employee`, поэтому попадают в module group автоматически.

Ключевые операции: workspace search, create/confirm operator order, open order support, add note, record adjustment, create escalation, supervisor escalation list. Все предопределенные сообщения представлены mnemonic-кодами `STR_MNEMO_EMPLOYEE_*`.
